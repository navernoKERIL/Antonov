# Теория

## Термины и определения

*   **Частота дискретизации:** Количество измерений в секунду (Гц).
*   **Период измерений:** Время между измерениями (с).
*   **Шаг квантования:** Минимальное изменение напряжения, которое может быть измерено АЦП (В).
*   **АЦП:** Преобразователь аналогового сигнала в цифровой.

## Физическая система

RC-цепь (резистор и конденсатор). Процесс заряда и разряда описывается экспоненциальными функциями:

Заряд: \[ V_c(t) = V_0 (1 - e^{-t/RC}) \]

Разряд: \[ V_c(t) = V_0 e^{-t/RC} \]

где:

*   \( V_c(t) \) – напряжение на конденсаторе.
*   \( V_0 \) – приложенное напряжение.
*   \( t \) – время.
*   \( R \) – сопротивление.
*   \( C \) – емкость.
*   \( RC \) – постоянная времени.

## Экспериментальная установка

*   Raspberry Pi
*   Тройка-модуль (ЦАП/АЦП)
*   RC-цепь (R = \[значение], C = \[значение])
*   Светодиоды

Пример кода для построения графика (Python):

```python
import matplotlib.pyplot as plt
import numpy as np

data = np.loadtxt('data.txt')
with open('settings.txt', 'r') as f:
    sampling_rate = float(f.readline())
    quantization_step = float(f.readline())

voltage = data * quantization_step
time = np.arange(0, len(voltage) / sampling_rate, 1 / sampling_rate)

plt.plot(time, voltage)
plt.xlabel('Время (с)')
plt.ylabel('Напряжение (В)')
plt.title('Заряд и разряд RC-цепи')
plt.grid(True)
plt.savefig('rc_charge_discharge.png')
plt.show()
